class Mod(x, n)[source]¶

Bases: object

An element x of ℤₙ.

Note

This class dispatches to one of RawMod, GMPMod or FlintMod based on what packages are installed and what implementation is configured (see pyecsca.misc.cfg).

Has all the usual special methods that upcast integers automatically:

>>> a = mod(3, 5)
>>> b = mod(2, 5)
>>> a + b
0
>>> a * 2
1
>>> a == 3
True
>>> a == -2
True
>>> -a
2

Plus some additional useful things:

>>> a.inverse()
2
>>> a.is_residue()
False
>>> (a**2).is_residue()
True
>>> (a**2).sqrt() in (a, -a)
True
